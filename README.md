# 🗳️ Democracy Guide: Your Interactive Election Assistant

Welcome to the **Democracy Guide**, a modern, interactive, and highly visual web application designed to educate and empower voters. This project aims to simplify the voting process through engaging tools and clear information.


## 🚀 Features

- **Interactive Learning**: Multi-tab navigation for seamless exploration of voter resources.
- **Voter Knowledge Quiz**: Test your understanding of the election process with an interactive quiz featuring real-time feedback.
- **Personalized Checklist**: A persistent voter checklist (saves to local storage) to ensure you're ready for election day.
- **Election Timeline**: A beautifully styled vertical timeline highlighting key election milestones.
- **Step-by-Step Voting Guide**: Clear, numbered instructions on how to register and cast your vote.
- **Smart FAQ**: An accordion-style FAQ section with built-in search functionality to find answers quickly.
- **Premium Design**: Built with a "wow" factor featuring glassmorphism, smooth animations, and a responsive mobile-first layout.

## 🛠️ Built With

- **HTML5**: Semantic structure for accessibility and SEO.
- **CSS3 (Vanilla)**: Advanced styling using variables, flexbox, grid, and keyframe animations.
- **JavaScript (Vanilla)**: Robust logic for state management, persistence, and interactive components.
- **Google Fonts**: Modern typography (Outfit & Inter).
- **FontAwesome**: High-quality vector icons.
